# WirePenguin
A CLI Packet Sniffer in golang using [Cobra](https://github.com/spf13/cobra) and [Gopacket](https://github.com/google/gopacket) packages much much simpler than WireShark =)  
## Commands
#### list
Lists all available network devices 
```
WirePenguin list
```
